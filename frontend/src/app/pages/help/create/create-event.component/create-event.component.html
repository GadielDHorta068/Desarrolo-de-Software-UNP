¿Cómo crear mi primer evento?

Sigue estos pasos simples para crear tu primer evento en RAFFIFY

  1. 1

    Inicia sesión o crea tu cuenta

    Accede a RAFFIFY con tu email o crea una nueva cuenta en segundos.

    Inicar sesión
  2. 2

    Autentificación de dos pasos

    Para poder crear un eventos, se requiere autentificación de dos pasos.

    Configurar autentificación
  3. 3

    Pasos para crear un evento

    1. 1

      Dirigete a tú perfil de usuario en "Mi Perfil".

    2. 2

      Ve al apartado "Eventos".

    3. 3

      Una vez dentro de Eventos haz click en "Nuevo Evento".

    4. 4

      Llena los campos (los que tienen * son obligatorios).

      Al seleccionar el "Tipo de evento" se mostrarán opciones específicas para ese tipo.

  4. 4

    Publica y comparte tu sorteo

    Publica el sorteo y comparte el enlace con tu audiencia.
